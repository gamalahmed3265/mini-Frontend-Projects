# Cyberpunk Walker Demo

A futuristic CSS/HTML walker animation with improved leg mechanics and a fixed head anchor.

## Features

- Realistic leg motion using a pivot chain: hip → thigh → knee → shin → foot.
- Head remains attached cleanly to the walker body and does not inherit torso animation.
- Hyperdrive mode toggled with `Space` speeds up visuals and updates the HUD.
- Moon click triggers a starburst animation effect.
- Head click briefly flashes the cyber eyes.

## How to run

1. Open `index.html` in your browser.
2. Press `Space` to toggle Hyperdrive.
3. Click the moon to trigger the starburst effect.
4. Click the head to flash the eyes.

## Notes

- This demo is designed as a single-page animation without external dependencies.
- The walker is rebuilt around a proper leg pivot hierarchy for smoother, more natural motion.
